PLC Automation Project

Smart Elevator PLC Simulator

Developed a Rockwell CCW elevator simulator using ladder logic, state-machine structure, floor request handling, and target floor selection.

Add main visual: screenshot of simulator, ladder logic, or state diagram.

Objective

With the goal of learning more about PLC design I wanted to build an elevator controller capable of responding to multiple floor requests and tracking the current floor.

My Contribution

  • Built the ladder logic structure in Rockwell Connected Components Workbench.
  • Created global/controller variables for floor requests and elevator state.
  • Implemented motor direction logic for up/down travel.
  • Implemented E-Stop feature to allow for stopping and returning to current state.
  • Added stop conditions based on current floor and active requests.
  • Reworked the program into a state-machine style structure.

Technical Details

  • Platform: Rockwell CCW simulator
  • Controller: Micro800 simulator environment
  • Logic: ladder diagram
  • Concepts: scan cycle, timers, latches, floor request memory, state transitions

Results

The final program operated as a working elevator state machine across multiple scripts, with request memory and floor-based stopping behavior.
